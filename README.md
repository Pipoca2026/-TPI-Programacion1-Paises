# Gestión de Datos de Países - TPI Programación 1

Este proyecto es una aplicación de consola desarrollada en Python para la gestión de información sobre países. Permite realizar operaciones de filtrado, ordenamiento y estadísticas sobre un dataset almacenado en formato CSV.

## Funcionalidades
- **ABM:** Agregar nuevos países y actualizar datos de población o superficie.
- **Búsqueda:** Localización de países por coincidencia parcial de nombre.
- **Filtros:** Por continente, rangos de población y superficie.
- **Ordenamiento:** Algoritmo de selección manual por nombre, población o superficie.
- **Estadísticas:** Cálculo de promedios, máximos, mínimos y conteo por continente.

## Requisitos
- Python 3.x
- No requiere librerías externas (usa módulos estándar como `csv` y `os`).

## Instrucciones de Uso
1. Clonar el repositorio.
2. Asegurarse de que el archivo `paises.csv` esté en la misma carpeta que el código.
3. Ejecutar el programa: `python CODIGOTPI.py`
